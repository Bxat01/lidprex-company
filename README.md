# Lidprex™

Lidprex™ is an independent software tools studio focused on building innovative developer tools and experimental tech platforms.

Founded in 2026 by an independent development team, Lidprex serves as the parent brand behind the following projects:

- **Idea2Project** – A platform that transforms ideas into organized, actionable projects for developers, freelancers, and startups.
- **RepoPrep** – Automatically cleans and organizes code repositories, removes unused dependencies, and suggests optimal folder structure.
- **LeakShield** – Offline security tool that scans projects and executables for leaks, secrets, misconfigurations, and vulnerabilities, providing actionable guidance.
- **MGS‑GenX** – Ambitious full-featured movie streaming platform (like Netflix / Prime Video). Requires proper licensing for content distribution; currently a project under Lidprex.

All products and tools under the Lidprex brand are developed and maintained by the Lidprex team.
